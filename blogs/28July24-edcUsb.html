I Always Carry a PC In My Pocket

28 July 2024


Like many, I always have my phone on me, but sometimes I need to use a computer that isn’t mine. Whether it’s checking a flight status while traveling, downloading an important document on a colleagues computer, or using a computer with outdated antivirus, my USB flash drives are my go-to solution for comfort and security.

I carry two USB flash drives: a 128GB flash drive on my key chain and a 64GB flash drive shaped like a credit card in my wallet.


Key Chain Flash Drive

My 128GB USB 3.0 key chain flash drive is durable and stores:

  • A folder of personal identification and important documents (encrypted with VeraCrypt)
  • A suite of portable apps for Windows (.exe) and Linux (.AppImage)
  • Various Linux distributions for live booting
  • General file storage and transfer

Encrypted Folder

I keep copies of my license, passport, and other important documents in an encrypted folder using VeraCrypt. VeraCrypt is a free, open-source program for creating encrypted drives, available for Windows, Mac, and Linux. It’s crucial to keep a copy of VeraCrypt on the flash drive for access on any computer.


Portable Apps

Portable apps run without installation, providing security and convenience on any system. I use portableapps.com for free downloads, including:

- 7zip, Audacity, Rufus, Solitaire, Firefox, Foxit, Calibre, GIMP, VIM, LibreOffice, VLC, and VeraCrypt

These apps are .exe files for Windows, but I also have .AppImage files for Linux, as some distributions lack Wine, which is needed to run .exe files on Linux. For Mac, I usually boot into a live Linux distribution.


Live Linux Distributions

I prefer Linux for its speed, power, and versatility. I carry:

  • Fedora-KDE for general use
  • Archbang for low-power modern machines
  • Puppy Linux for old machines
  • Kali Linux for network and security tools

I can boot into a full version of any of these distributions on any available computer right from the USB.

Ventoy helps me manage multiple .iso files and other files on a single bootable flash drive.


General File Storage and Transfer

Even with all the above, I use less than half the space on my key chain flash drive, leaving plenty for general file storage and transfer.


USB Credit Card Flash Drive

For times when I don’t have my keys, I carry this 64GB credit card-shaped flash drive in my wallet. It’s rarely used but there if needed.


Conclusion

Having a USB flash drive ensures secure access to digital files, software, and the internet. My setup handles about 99% of situations, giving me confidence.
